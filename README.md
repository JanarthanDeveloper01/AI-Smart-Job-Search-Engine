# 💼 AI Smart Job Search Engine

An AI-powered web application that analyzes a user's resume and recommends relevant job roles along with job links from platforms like LinkedIn, Indeed, and Naukri.

---

## 🚀 Features

- 📄 Resume Upload (PDF / DOCX)
- 🤖 AI-based Resume Analysis using Gemini API
- 🎯 Automatic Job Role Detection
- 🔁 Hybrid AI + Rule-based fallback system
- 🔎 Job Recommendations for multiple roles
- 🔗 Direct job links (LinkedIn, Indeed, Naukri)
- 📊 Resume vs Job Match Score

## 🧠 Tech Stack

- **Frontend:** Streamlit
- **Backend:** Python
- **AI:** Google Gemini API
- **Libraries:**
  - pdfplumber
  - python-docx
  - scikit-learn
  - dotenv
